BOSH task¶

BOSH 를 통한 모든 작업은 task 라고 하는 단위로 관리 된다. 그리고 task 를 조회하는 작업도 하나의 task 이므로 아래와 같이 watch 명령으로 반복 조회를 할 경우 어느 순간 task number 가 순식간이 올라가 있는 것을 발견하게 될 것이다. 조심하자.

cancel-task¶

    $ bosh -e lite cancel-task 256
